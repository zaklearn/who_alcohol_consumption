  • Access Method: WHO GHO OData API

üìè Measurement Definitions

Alcohol Consumption (APC)

Definition: Recorded per capita (15+ years) consumption in litres of pure alcohol

  • Based on recorded production, import, export, and sales data
  • Often derived from taxation records
  • Three-year averages calculated (e.g., 2015 = average of 2014, 2015, 2016)
  • Excludes unrecorded consumption (home-made, illegally produced)

Alcohol Use Disorders

Definition: Adults (15+ years) with disorders attributable to alcohol consumption

  • Based on ICD-10 classifications: F10.1 (Harmful use) and F10.2 (Dependence)
  • Data modeled using regression models
  • Prevalence expressed as percentage of population
  • Disaggregated by gender (Male, Female, Both sexes)

üõ†Ô∏è Data Processing Methods

Data Extraction

  • API calls to WHO GHO OData endpoints
  • Country name standardization for consistent merging
  • Missing value imputation (2005 values imputed with 2004 data where missing)

Analysis Techniques

  • Descriptive Statistics: Mean, median, range calculations by region
  • Time Series Analysis: Trend identification over 22-year period
  • Correlation Analysis: Pearson correlation between consumption and disorders
  • Linear Regression: Relationship modeling with R-squared calculation

Visualization Methods

  • Interactive Charts: Plotly library for dynamic visualizations
  • Geographic Mapping: Choropleth maps for spatial analysis
  • Statistical Plots: Scatter plots with regression lines
  • Time Series Plots: Multi-country trend comparisons

‚ö†Ô∏è Limitations and Considerations

Data Limitations

  • Coverage: Not all countries have complete data for all years
  • Unrecorded Consumption: Home-made and illegally produced alcohol not captured
  • Cultural Factors: Religious and cultural prohibitions may affect reporting
  • Economic Factors: Informal markets may not be fully captured

Analytical Considerations

  • Correlation vs Causation: Statistical relationships don't imply causal mechanisms
  • Temporal Alignment: Different indicators may have different reference periods
  • Regional Grouping: WHO regional classifications may not reflect all cultural similarities
